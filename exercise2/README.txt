Setting a polynomial by using a vector, the vector store all coefficients from the highest order to the lowest.

Constructor:
Polynomial();
Polynomial(vector<double> coefficients);

Destructor:
~Polynomial();

Function:
get_degree(): Return an integer which is the degree of the polynomial
print(): Print the polynomial function P(x) from the highest order to the lowest

Variables:
_coefficients: A vector, store all coefficients from the highest order to the lowest
_degree: A integer, the degree of the polynomial
